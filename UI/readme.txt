# Project Set Up

## Pre-Requirements

- Install NodeJS
- Go to folder where package.json file is located
- Install all node dependencies running in command line `npm install`

## Local Development

- Build the app and watch for changes running in command line `npm start`
- Go to http://localhost:3000/

## Development build

- Build the app running in command line `npm run build:dev`
- Packaged files are outputed in './builds' folder.
- Copy files to to respective server location.

## QA build

- Build the app and minify code running in command line `npm run build:qa`
- Packaged files are outputed in './builds' folder.
